A **Music Center** Software designed in **Frutiger Aero (Dark / Aurora)** style.

## Key Mappings

### Default Keys

| KEY0 | KEY1 | KEY2 | KEYL | KEYM | KEYR |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 1    | 2    | 3    | 4    | 5    | 6    |

### Page Keys

| Page   | KEY0 | KEY1   | KEY2       | KEYL          | KEYM       | KEYR      |
| ------ | ---- | ------ | ---------- | ------------- | ---------- | --------- |
| Menu   | -    | Player | Confirm    | Previous      | -          | Next      |
| Album  | Back | Player | Confirm    | Previous      | Play Album | Next      |
| Player | Back | -      | Play/Pause | Previous Song | -          | Next Song |

## Format Support

- MP3
- MKV/WebM
- OGG
- WAVE
- FLAC
